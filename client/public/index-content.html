🔒 4つの実行モード

Claude Code がファイルを編集・コマンドを実行するとき、どこまで自動で進めるかを4段階で制御できる。右下の盾アイコンから切り替え可能。

🔴
許可を確認 （デフォルト）
ファイル編集・コマンド実行のたびに確認ダイアログが出る。最初はこのモードで始めよう。何をしているか把握しながら進められる。
🟡
編集を自動承認
ファイルの読み書きは自動で進む。コマンド実行（ターミナル操作）だけ確認が入る。慣れてきたら使うと作業がスムーズになる。
🟣
プランモード
実際には何もせず、「こういう手順でやります」という計画だけを提示するモード。大きな変更の前に「どう進めるか確認したい」ときに便利。
🟢
許可をバイパス
すべての操作を確認なしで自動実行する。慣れた作業・信頼できるプロジェクトで使う上級者向けモード。最初は使わないこと。
💡 おすすめの使い方 最初は「許可を確認」で始めて、Claudeが何をしているか確認しながら慣れよう。慣れてきたら「編集を自動承認」に切り替えると快適になる。
